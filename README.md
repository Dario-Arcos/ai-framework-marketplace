# AI Framework Marketplace

Curated Claude Code plugins for AI-first development.

## Installation

```bash
# Add this marketplace to Claude Code
/plugin marketplace add Dario-Arcos/ai-framework-marketplace

# Install the AI Framework plugin
/plugin install ai-framework@ai-framework-marketplace
```

## Available Plugins

| Plugin | Description | Version |
|--------|-------------|---------|
| [ai-framework](https://github.com/Dario-Arcos/ai-framework) | Constitutional AI-first development framework with specialized agents, workflow commands, and intelligent hooks | 4.3.1 |
